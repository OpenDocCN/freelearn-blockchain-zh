## 下载

### Docker

```
docker pull apachecn0/freelearn-blockchain-zh
docker run -tid -p <port>:80 apachecn0/freelearn-blockchain-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-blockchain-zh
freelearn-blockchain-zh <port>
# 访问 http://localhost:{port} 查看文档
```
